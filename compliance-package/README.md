# Pakiet zgodności (Compliance Pack)

Pakiet zawiera komplet materiałów wymaganych do uruchomienia podstawowego programu compliance:

1. **Polityki i dokumenty prawne**
   - `01-polityka-prywatnosci.md`
   - `02-regulamin.md`
   - `03-dpa.md`
   - `04-cookie-policy.md`
2. **ROPA i data mapping**
   - `05-rejestr-przetwarzania-i-mapowanie.md`
3. **Procedury bezpieczeństwa**
   - `06-procedury-bezpieczenstwa.md`
4. **Checklisty zgodności**
   - `07-checklista-zgodnosci.md`
5. **Trust Center**
   - `../trust-center/trust-center-section.md`

## Jak używać
- Uzupełnij placeholdery (`[NAZWA FIRMY]`, `[DOMENA]`, `[E-MAIL]`, itp.).
- Zweryfikuj dokumenty z działem prawnym przed publikacją.
- Ustal ownerów i harmonogram przeglądów kwartalnych.
- Publikuj aktualne wersje dokumentów w jednym miejscu (Trust Center).
