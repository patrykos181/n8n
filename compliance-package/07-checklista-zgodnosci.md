# Checklista zgodności: RODO/GDPR, SOC 2-ready, ISO 27001-ready

**Wersja:** 1.0  
**Data:** 30 marca 2026 r.

## 1. GDPR / RODO
- [ ] Zidentyfikowane role (administrator/procesor/współadministrator).
- [ ] Aktualny rejestr czynności przetwarzania (ROPA).
- [ ] Podstawy prawne przetwarzania przypisane do procesów.
- [ ] Mechanizmy realizacji praw osób (DSAR).
- [ ] Procedura zgłaszania naruszeń i ocena 72h.
- [ ] Umowy powierzenia (DPA) z podmiotami przetwarzającymi.
- [ ] Transfery poza EOG zabezpieczone (SCC/TIA).
- [ ] Privacy by design/by default w procesie wytwarzania.
- [ ] Retencja danych i bezpieczne usuwanie.
- [ ] Szkolenia personelu dot. ochrony danych.

## 2. SOC 2-ready (Trust Services Criteria)
- [ ] Security: kontrola dostępu, MFA, hardening, logowanie.
- [ ] Availability: monitoring, backupy, test odtwarzania.
- [ ] Confidentiality: klasyfikacja danych, szyfrowanie, need-to-know.
- [ ] Processing Integrity: kontrola zmian i testy jakości.
- [ ] Privacy: polityki prywatności, zgody, ograniczenie celu.
- [ ] Formalny rejestr ryzyk i plan mitigacji.
- [ ] Dowody kontroli (audit trail) dostępne i aktualne.

## 3. ISO 27001-ready
- [ ] Zdefiniowany ISMS i zakres certyfikacji.
- [ ] Metodyka oceny ryzyka i statement of applicability (SoA).
- [ ] Polityki bezpieczeństwa zatwierdzone przez kierownictwo.
- [ ] Kontrole organizacyjne, fizyczne i techniczne wdrożone.
- [ ] Zarządzanie incydentami i ciągłością działania (BCP/DR).
- [ ] Zarządzanie aktywami i klasyfikacją informacji.
- [ ] Program audytów wewnętrznych i przegląd zarządzania.
- [ ] Plan ciągłego doskonalenia (CAPA).
