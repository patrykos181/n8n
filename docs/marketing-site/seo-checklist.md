# SEO technical checklist

## Global
- [ ] `title` 50–60 znaków i unikalny per URL.
- [ ] `meta description` 140–160 znaków.
- [ ] `canonical` ustawiony dla każdego URL.
- [ ] Open Graph (`og:title`, `og:description`, `og:image`, `og:url`).
- [ ] Twitter Card (`summary_large_image`).

## Indexation
- [ ] `robots.txt` dostępny pod `/robots.txt`.
- [ ] `sitemap.xml` dostępny pod `/sitemap.xml`.
- [ ] Strony stagingowe zablokowane (`Disallow: /`).
- [ ] Brak indexacji stron thin/duplicate.

## Schema.org
- [ ] `Organization` na stronie głównej.
- [ ] `WebSite` z `SearchAction`.
- [ ] `BreadcrumbList` na stronach zagłębień.
- [ ] `Article` na blogu.
- [ ] `FAQPage` tam gdzie widoczny FAQ.

## CWV
- [ ] LCP mobile < 2.5s (P75).
- [ ] INP < 200ms (P75).
- [ ] CLS < 0.1 (P75).
- [ ] Hero media z rozmiarem i priorytetem ładowania.
- [ ] Third-party scripts załadowane asynchronicznie/po interakcji.

## Content hygiene
- [ ] 1x H1 per page.
- [ ] Sensowna hierarchia H2/H3.
- [ ] Linkowanie wewnętrzne do money pages.
- [ ] FAQ zawiera pytania z intencji zakupowej.
